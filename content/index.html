--- title: R --- R'adas ===================== Exemplos de análises estatísticas feitas em R ------------ Vou jogando aqui códigos comentados de análises estatísticas que vou fazendo, seja elas para casos reais ou não. O conteúdo do site supõe que o leitor tenha certa familiaridade com estatística. Conhecimento sobre o basicão do R ajuda, mas não precisa, vou colocando coisas sobre o elementar de R também. Os tópicos de estatística que existem até o momento são: * Análise Descritiva * Análise Multivariada * Modelos Lineares * Modelos Aditivos * Inferência Bayesiana * Regressão Logística * Processos Estocásticos Coisas em R além da análise de dados ---------------- O R é muito bom em fazer aquilo que inicialmente lhe foi proposto a fazer. Mas a verdade é que dá para fazer muito mais com ele, inclusive tarefas que otimizam/aprimoram as produções de um estatístico. O campo a ser explorado é vasto! Tem uma seção de códigos só para isso. Os tópicos arranhados até o momento foram: * Gráficos com ggplot2 * Aplicações Web com Shiny * Envio de e-mails * R markdown * Manipulação de Banco de dados com dplyr * Web Scrapping * Datas com lubridate Open Source ------------------- Todos os códigos desse site foram feitos em `.Rmd` (R markdown) e estão disponíveis no meu Github. Isso significa que além de todos poderem reaproveitar os códigos, poderão melhorá-los, complementá-los e até fornecer um `.Rmd` totalmente novo para ser incluído aqui. Todo mundo ganha!
